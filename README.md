# Cinsiyet Tahmini Uygulaması

Bu proje, JavaScript dersinde **Fetch API** konusunu öğrenmek amacıyla hazırlanmıştır.

Kullanıcıdan alınan isim bilgisi, **Genderize API** kullanılarak analiz edilir ve
ismin cinsiyet tahmini ekranda gösterilir.

## Kullanılan Teknolojiler
- HTML
- CSS
- JavaScript
- Fetch API

## Uygulama Özellikleri
- İsim girilerek cinsiyet tahmini yapılır
- Sonuçlar kart şeklinde listelenir
- Detay butonu ile ek bilgiler gösterilir
- Sayfa içi hakkımızda ve iletişim bölümü bulunur

## API
- https://api.genderize.io/?name=alex
