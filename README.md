# Cinsiyet Tahmini

Bu proje JavaScript dersinde Fetch API konusunu öğrenmek için yapılmıştır.

Kullanıcıdan alınan isme göre API üzerinden cinsiyet tahmini yapılır.
Gelen bilgiler JavaScript ile ekranda gösterilir.

---

## Kullanılan API

Genderize API

Örnek kullanım:
https://api.genderize.io/?name=alex

---

## Uygulama Nasıl Çalışır?

- Kullanıcı isim girer
- Ara butonuna basar
- API’den veri çekilir
- Sonuçlar ekranda gösterilir
- Detay butonu ile detay alanı açılır

---

## Kullanılan JavaScript Konuları

- Fetch API
- try / catch
- addEventListener
- if / else
- Fonksiyon kullanımı
- forEach
